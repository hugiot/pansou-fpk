### 安装

1. 通过 [Releases 页面](https://github.com/hugiot/pansou-fpk/releases) 下载最新版本的安装包（pansou.fpk）；
2. 打开飞牛应用中心，选择手动安装；
3. 选择 “从电脑上传” 并选择下载好的 pansou.fpk 文件；
4. 安装完成后，在应用中心找到 pansou 应用，点击启动即可。
